Instruções de compilação: 
A compilação é realizada utilizando o comando "gpp *.c -o <nomeDoExecutavel>"
A execução pode ser realizada de duas formas:
(1) -> utilizando o comando ".\<nomeDoExecutavel> <nomeDoArquivo>", onde o segundo parametro pode ser um arquivo texto para realizar o cadastro prévio dos itens.
(2) -> utilizando o comando ".\<nomeDoExecutavel>", onde todas as inserções devem ser manuais.
